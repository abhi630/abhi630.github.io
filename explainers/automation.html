AI Automation

The full spectrum from manual to agentic — what changes before and after AI automation, and how to build it right.

L1: Manual

A human performs every step by hand. No automation at all.

Example: Copy-pasting rows from one spreadsheet to another

Uses AI
No

L2: Scripted

Deterministic code runs the same steps every time — no judgment required.

Example: A cron job that runs a SQL query and emails the result at 9am

Uses AI
No

L3: Rule-based AI

AI applies learned rules to variable inputs — classification, routing, filtering.

Example: Spam filter, sentiment classifier, intent router in a chatbot

Uses AI
Yes

L4: Generative AI

AI produces variable outputs — text, code, decisions — given variable inputs.

Example: Draft customer reply, write a report section, generate SQL from natural language

Uses AI
Yes

L5: Agentic AI

AI takes multi-step actions, uses tools, loops until a goal is reached — with minimal human direction.

Example: Research a topic, write a briefing, send it to stakeholders — all in one run

Uses AI
Yes

Most organisations live at L1-L2. The jump to L4-L5 (generative + agentic) is where 10x productivity gains become available — but also where design and oversight matter most.

▼
1. Automate the repeatable
If a task follows the same steps every time and quality is measurable, it's a good automation candidate. If every instance requires fresh judgment, it's not.
▼
2. Start with output, not process
Don't try to replicate how a human does a task. Define what a good output looks like, and let the AI find a route to it.
▼
3. Keep a human in the loop for consequences
Any automation that sends emails, modifies files, or makes purchases should have a human review step unless the stakes are very low.
▼
4. Measure quality explicitly
Automate only what you can evaluate. If you can't tell whether the output is good, you can't trust the automation.
▼
5. Build the simplest thing first
Start with a prompt and a manual trigger. Add scheduling, tools, and error handling only when you've validated the core output.
