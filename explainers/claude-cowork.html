What is Claude Cowork?

The desktop app that brings Claude into your actual file system — with skills, scheduling, and real deliverables.

Claude Cowork is a desktop app that brings Claude into your actual workflow — with access to your files, the ability to run scheduled tasks, and skills pre-built for specific knowledge work.

vs Claude.ai

Claude.ai is a conversation. Cowork is a co-worker. It opens your files, runs on a schedule, uses specialist skills, and produces real deliverables — not just chat responses.

The Core Idea

Instead of constantly copying content into a chat box, you give Claude access to your folder, describe the task, and it handles the rest — reading, processing, writing, saving.

The most powerful thing about Cowork isn't a single feature — it's the combination: file access + skills + scheduling means Claude can handle entire recurring workflows, not just one-off tasks.

File Access

Claude reads and writes real files on your computer — Word docs, spreadsheets, PDFs, code files — not just copy-pasted text.

Why it matters: Stop copy-pasting content into a chat box. Grant Cowork access to a folder and Claude works directly with your files.

Example: "Summarise all PDFs in this folder and write a report"

Skills & Plugins

Skills are pre-built instruction sets that prime Claude for specific workflows — presentations, spreadsheets, marketing content, legal review.

Why it matters: Instead of writing complex prompts every time, you invoke a skill and Claude immediately knows the format, style, and standards.

Example: "/pptx Create a 10-slide pitch deck from this brief"

Scheduled Tasks

Cowork tasks can run on a cron schedule — daily, weekly, at a specific time — without you being present.

Why it matters: Automate repetitive knowledge work: daily briefings, weekly reports, regular content generation.

Example: "Every Monday 9am: pull last week's metrics and draft summary"

Desktop App

Cowork runs as a native desktop application alongside your other tools — not a browser tab you lose track of.

Why it matters: Persistent access to your files, no browser permission friction, feels like part of your OS.

Example: Sits in your dock. Open, give a task, close. Work happens.

Task Tracking

Cowork shows a live todo list as Claude works through multi-step tasks — so you always know what it's doing.

Why it matters: Long tasks become transparent. You can see "Step 3 of 7" rather than wondering if Claude is still thinking.

Example: Progress bar with labelled steps updates in real time

▼
1. Grant folder access
You choose which folder on your computer Cowork can read and write. Everything inside that folder is accessible to Claude. You control exactly what Claude can see. One project folder, a downloads folder, a specific workspace — your choice. Nothing outside that folder is accessible.
▼
2. Type a task
In plain language: "Read all the PDFs here and write a 1-page summary", or invoke a skill with /docx. Skills like /pptx, /docx, /pdf load expert-level instructions so Claude knows exactly how to format and structure the output.
▼
3. Watch the todo list
Cowork shows each step it's taking — reading File A, processing File B, writing output. You see exactly what's happening. The todo list updates in real time. For a 10-file task you see each file tick off. You can read the chat, ask questions, or step away.
▼
4. Find your output
The finished file appears in your outputs folder, ready to open. Cowork links to it directly in the chat. Cowork outputs land in a dedicated folder with a clickable link. For supported formats (.docx, .pptx, .pdf) you can open them directly.
